# OSMBadgeStages
Places the stage/level of a badge below a badge in the Badges by Person page

Also has logic to display stage of alternative Membership badge (Squirrel,Beaver,Cub,Scout,Explorer,Network,Adult) using badge created under 23rd Mancheter Group.
